# About Graph Algorithms

This is the place where I gather my understanding of graph algorithms and
theory.

Recently it came to my hands the book: **Graph Algorithms The Fun Way**,
*Powerful Algorithms Decoded, not oversimplified*,  by Jeremy Kubica
[ISBN: 978-1-7185-0386-1] © 2025; and decided to give *graphs* another try to
understand them.

Previously, I had tried with:

- **Algorithms in C++**, *Part 5: Graph Algorithms*, by Robert Sedgewick and
Christopher J. Van Wyk [ISBN: 0-201-36118-3] 3rd edition © 2002.
- **Introduction to Algorithms**, by Thomas H. Cormen, Charles E. Leiserson,
Ronald L. Rivest [ISBN: 0-262-03141-8] © 1990.
- **Introduction to Graph Theory**, by Richard J. Trudeau
[ISBN: 978-0-486-67870-9] @ 1993

I'm making more way with the last book '*... the fun way*', than with the
others; so, I decided to record my progress.

The '*... the fun way*' is written with code in Python, so Python 3.12 it is;
why 3.12 ... mostly because most of my other pieces of code in Python at the
time of this writing is written using 3.12 (some day in the future this lines
are going to look old an obsolete, and it will be true then 😉).
